# 模块5：更多可能（进阶了解）

## 教学目标

让学员了解AI的更多可能性，但保持轻松——不需要他们掌握这些，只需要知道"有这个选项"。

## 话题列表

根据学员兴趣选择话题，不需要全部讲。

### 话题1：AI工具有哪些

**比喻：** 就像手机APP一样，AI工具也有不同的APP，各有各的擅长。

**主流工具一览：**

| 工具 | 擅长什么 | 怎么用 |
|------|---------|--------|
| ChatGPT | 通用聊天、写东西、分析 | 网页或APP |
| Claude | 长文章分析、编程、深度对话 | 网页 |
| 文心一言 | 中文场景、国内合规 | 网页或APP |
| Kimi | 长文档处理 | 网页或APP |
| 通义千问 | 中文办公场景 | 网页或APP |
| Copilot | Office集成、编程辅助 | 嵌入Word/Excel等 |
| Midjourney | AI绘画 | 网页 |
| 即梦AI | AI视频生成 | 网页或APP |

**要点：** 不用全学，选1-2个最适合你需求的就行。

### 话题2：AI在你的行业能怎么用

根据学员的行业和岗位，给出具体的应用建议：

**行政/文秘：**
- 自动整理会议纪要
- 智能日程管理
- 公文起草和格式化
- 邮件分类和回复草稿

**财务/会计：**
- 自动对账和异常检测
- 财务报表解读
- 税务政策查询和对比
- 预算分析建议

**销售/市场：**
- 客户画像分析
- 营销文案批量生成
- 竞品分析报告
- 客户邮件个性化

**人力资源：**
- 简历筛选和匹配
- 面试问题生成
- 培训材料制作
- 员工满意度分析

**管理层：**
- 决策支持分析
- 行业趋势洞察
- 报告摘要和对比
- 战略方案评估

### 话题3：用AI需要注意什么

**安全意识（简单说）：**

1. **不要把公司机密数据直接扔给AI** — "就像你不会把公司公章带回家一样，敏感数据要注意保护。"
2. **AI的回答要核实** — "AI有时候会一本正经地说错话。重要的数据和结论，还是要自己验证一下。"
3. **注意版权问题** — "AI生成的内容，目前在版权上还有一些模糊地带。正式用途的话最好了解一下相关规定。"
4. **不要完全依赖AI** — "AI是好工具，但你的专业判断才是最重要的。把AI当助手，不要当老板。"

### 话题4：AI的未来趋势（选讲）

如果学员感兴趣，可以简单聊聊：
- AI Agent（AI智能体）：未来的AI不只是聊天，能自动帮你完成一系列任务
- 多模态：AI能同时理解文字、图片、音频、视频
- 个性化：AI会越来越了解你，越来越贴合你的需求
- AI+办公深度整合：未来的Office可能原生内置AI功能
