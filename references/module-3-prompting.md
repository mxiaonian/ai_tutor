# 模块3：怎么跟AI说话（提示词技巧）

## 教学目标

让学员学会用自然语言有效地与AI沟通。核心理念：跟AI说话就像跟一个聪明但完全不了解你背景的新同事说话——你需要交代清楚背景和需求。

## 核心理念

**一句话总结：** 跟AI说话的关键不是学什么"咒语"，而是学会把你的需求说清楚。

**比喻：** 你带一个新同事做项目。你不会跟他说"帮我搞一下那个东西"，你会说"小李，帮我把Q3的销售数据整理一下，做成一个饼图，重点突出华东区的数据，下午开会要用"。跟AI说话也是一样。

## 提示词的四个要素

用一个口诀帮助记忆：**"谁、做什么、怎么做、要什么"**

### 1. 角色设定（谁）
告诉AI它扮演什么角色，这会影响回答的角度和深度。

**例子：**
- "你是一个资深财务分析师……"
- "你是一个有10年经验的HR经理……"
- "你是一个小红书文案高手……"

### 2. 具体任务（做什么）
明确告诉AI你要它干什么。

**反面：** "帮我写个报告"
**正面：** "帮我写一份2024年Q3的市场推广效果分析报告，包含数据对比和下季度建议"

### 3. 格式要求（怎么做）
告诉AI你希望结果长什么样。

**例子：**
- "用表格的形式呈现"
- "分三个要点来回答"
- "控制在500字以内"
- "用通俗易懂的语言，不要用专业术语"

### 4. 背景信息（要什么）
补充必要的上下文。

**例子：**
- "我的受众是公司高管，他们只关心投入产出比"
- "我们是一家做母婴产品的电商公司"
- "我需要中文回复"

## 实用技巧

### 技巧1：不满意的就让它重来

"这个太正式了，能不能轻松一点？"
"字数太多了，精简到一半。"
"方向对了，但能不能更具体一点？"

AI不会因为你说"重做"而不高兴。这是它和人类最大的区别。

### 技巧2：给例子

如果你想要特定风格，直接给它一个例子：
"我想要类似这种风格的文案：【粘贴一段你喜欢的文案】"

### 技巧3：分步骤

复杂任务不要一句话说完，拆成几步：
1. "先帮我列出这篇文章的大纲"
2. "大纲不错，按这个大纲帮我写第一部分"
3. "继续写第二部分"

### 技巧4：让它问你

如果不知道怎么描述需求，可以说：
"我想做一个关于XX的东西，但我不太确定怎么表达。你能问我几个问题来帮我把需求弄清楚吗？"

## 练习环节

给学员几个场景，让他们试着写提示词：

**场景1：** 你需要写一封给客户的邮件，告知项目延期。
**场景2：** 你需要把一份英文报告翻译成中文，但要用公司内部的术语。
**场景3：** 你需要AI帮你分析一份Excel表格里的销售数据。

每个场景先让学员试着自己写，然后给出优化建议，对比效果。

## 讲解节奏

1. 先说："其实你已经会跟AI说话了——你之前跟我聊天的过程，就是在给AI写'提示词'"
2. 介绍四个要素
3. 做一个前后对比演示：模糊的提示词 vs 精确的提示词
4. 进入练习环节
5. 总结：关键是说清楚，不需要学什么特殊格式
