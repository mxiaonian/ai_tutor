# 模块2：AI能做什么

## 教学目标

让学员看到AI的实际能力，建立"这东西真的有用"的信心。重点是通过当场演示，而不是口头描述。

## AI的主要能力

### 1. 写东西

**比喻：** 就像一个写作速度极快、知识面很广的同事。你给它一个大纲，它能在几秒钟写出一篇像样的初稿。

**能做什么：**
- 工作邮件、周报、会议纪要
- 文档总结（扔给它一份20页的报告，它30秒给你摘要）
- 翻译（支持几十种语言）
- 文案、广告语、社交媒体内容

**建议演示：** 当场用 python-docx 生成一份"项目进度周报"或"会议纪要模板"。

### 2. 处理数据

**比喻：** Excel函数高手，而且是一个永远不会累、永远不会输错公式的函数高手。

**能做什么：**
- 数据清洗和整理
- 写Excel公式和VBA宏
- 数据分析和可视化
- 从一堆数据里找出规律

**建议演示：** 当场用 openpyxl 生成一个带公式、条件格式、图表的销售数据分析表。

### 3. 做PPT

**比喻：** 一个懂设计、会排版、内容还不错的演示文稿助手。

**能做什么：**
- 根据主题生成PPT大纲
- 自动排版和设计
- 生成演讲稿

**建议演示：** 当场用 python-pptx 生成一个"2024年度工作汇报"风格的PPT。

### 4. 写代码

**比喻：** 一个随叫随到的程序员。你用中文说需求，它帮你写代码。

**能做什么：**
- 自动化办公任务（批量处理文件、自动发邮件等）
- 数据爬取和处理
- 小工具开发

**建议演示：** 当场写一个简单的Python脚本并运行，比如"批量重命名文件"或"生成随机密码"。

### 5. 回答问题和学习

**比喻：** 一个百科全书式的导师，随时可以问。

**能做什么：**
- 解释任何概念（会用简单语言）
- 帮你学习新技能
- 出题测试你的理解

### 6. 图片相关

**能做什么：**
- 生成图片（DALL-E、Midjourney等）
- 识别图片内容
- 修图和处理

## 讲解节奏

1. 问学员："你平时工作中最花时间的事是什么？"
2. 根据回答，选择最相关的1-2个能力优先演示
3. 演示完问："你觉得这个对你有用吗？"
4. 再介绍其他能力

## 演示时的话术模板

- "我给你做一个实际的演示，你看看效果。"
- [执行操作，生成文件]
- "好了，我给你在桌面上生成了一个文件，叫 xxx.xlsx，你打开看看。"
- "你看这个表格，里面的公式、数据、图表都是自动生成的。如果你自己手动做这个，大概要多久？AI几秒钟就搞定了。"
- "当然，这还是个半成品，你需要根据实际情况调整。但它帮你把最费时间的部分搞定了。"
