# 模块4：AI帮你干活（Office实战演示）

## 教学目标

通过实际的Office文件演示，让学员亲眼看到AI能怎么帮他们处理日常工作。重点是"看到效果"，不需要学员理解背后的技术。

## 演示项目

### 演示1：AI帮你做Excel数据分析

**场景：** 你有一份销售数据，需要做分析。

**话术：** "假设你是销售经理，月底要交一份销售报告。我让AI帮你生成一份示例数据和分析表格，你看看效果。"

**生成步骤：**
1. 用 openpyxl 创建一个包含以下内容的 .xlsx 文件：
   - Sheet1: 原始销售数据（产品名、区域、月份、销售额、成本、利润等，至少30行真实感数据）
   - Sheet2: 数据透视表（按区域和月份汇总）
   - Sheet3: 关键指标卡片（总销售额、总利润、利润率、最佳产品等）
   - 包含基本图表（柱状图或饼图）
2. 添加公式（SUM、AVERAGE、百分比计算等）
3. 加上条件格式（高于目标的标绿，低于目标的标红）
4. 保存到 `~/Desktop/AI演示_销售数据分析.xlsx`

**做完后说：** "你打开桌面上的那个Excel文件看看。里面的数据、公式、图表都是AI自动生成的。如果这是你真实的销售数据，这份分析报告从零开始做，你觉得要多久？AI大概用了10秒。"

### 演示2：AI帮你写Word文档

**场景：** 会议结束，需要整理会议纪要。

**话术：** "开完会写纪要是不是很烦？来，我让AI帮你生成一个模板和示例内容。"

**生成步骤：**
1. 用 python-docx 创建一个 .docx 文件：
   - 标题：项目周会会议纪要
   - 基本信息：日期、地点、参会人员
   - 会议议题
   - 讨论内容摘要
   - 决议事项（表格形式）
   - 待办事项（含负责人和截止日期）
   - 下次会议安排
2. 使用专业格式（标题层级、表格、列表）
3. 保存到 `~/Desktop/AI演示_会议纪要模板.docx`

**做完后说：** "你看这个格式，是不是挺专业的？以后你开会只需要把要点发给AI，它帮你整理成这种格式。你只需要检查一下内容对不对就行。"

### 演示3：AI帮你做PPT

**场景：** 年底了，要做年度工作汇报。

**话术：** "年底汇报做PPT是不是头疼？我让AI帮你做一个框架出来。"

**生成步骤：**
1. 用 python-pptx 创建一个 .pptx 文件：
   - 封面页（标题、汇报人、日期）
   - 目录页
   - 年度业绩概览（关键数据）
   - 重点项目回顾（2-3页）
   - 团队成果展示
   - 问题与挑战
   - 下一年度计划
   - 感谢页
2. 每页有简要的要点文字
3. 使用简洁的配色
4. 保存到 `~/Desktop/AI演示_年度工作汇报.pptx`

**做完后说：** "这个PPT的骨架已经出来了。你只需要把真实数据填进去，调整一下内容。比从空白PPT开始，省了很多时间吧？"

### 演示4：AI帮你自动化

**场景：** 需要批量处理一堆文件或者做重复性操作。

**话术：** "你工作中有没有遇到过那种重复性特别强、但又要一个个弄的任务？比如给100个文件重命名，或者从一堆表格里提取数据？来看AI怎么搞定。"

**演示脚本示例（选择一个适合学员的）：**

```python
# 示例：批量生成工作周报文件
import os
from datetime import datetime, timedelta

desktop = os.path.expanduser("~/Desktop")
folder = os.path.join(desktop, "AI演示_批量周报")
os.makedirs(folder, exist_ok=True)

start_date = datetime(2024, 1, 1)
for i in range(12):  # 生成12周的周报文件夹
    week_start = start_date + timedelta(weeks=i)
    week_end = week_start + timedelta(days=4)
    folder_name = f"第{i+1}周_{week_start.strftime('%m%d')}-{week_end.strftime('%m%d')}"
    os.makedirs(os.path.join(folder, folder_name), exist_ok=True)
    # 在每个文件夹里创建一个周报模板
    with open(os.path.join(folder, folder_name, "周报.md"), "w") as f:
        f.write(f"# 第{i+1}周工作周报\n\n")
        f.write(f"## 本周工作 ({week_start.strftime('%m/%d')} - {week_end.strftime('%m/%d')})\n\n")
        f.write("1. \n\n")
        f.write("## 下周计划\n\n")
        f.write("1. \n")

print(f"已在桌面生成 {folder} ，包含12周的周报文件夹和模板！")
```

**做完后说：** "这个是AI写的一段小程序，几秒钟就帮你建好了12周的文件夹和模板。如果手动建，要点多少次新建文件夹？这就是AI编程的厉害之处——你不需要学编程，只需要知道AI能帮你做这种事。"

## 注意事项

- 生成文件前先安装需要的库（openpyxl、python-docx、python-pptx），默默安装，不要让学员看到复杂的安装过程
- 所有生成的文件放在 `~/Desktop/` 并以"AI演示_"开头命名，方便学员找到和清理
- 演示时重点强调"省了多少时间"，而不是"技术多厉害"
- 询问学员："你工作中有没有类似的事情？我可以给你定制一个。"
