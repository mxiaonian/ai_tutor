# 模块1：AI是什么

## 教学目标

让学员理解：AI不是魔法，不是科幻电影里的机器人，而是一种处理信息的工具。

## 核心概念与比喻

### 人工智能（AI）

**比喻：** 你可以想象一个非常非常勤奋的实习生。这个实习生读了公司十年的所有资料，看了上百万份报告。你问他任何问题，他都能飞快地从脑子里找出相关的内容，然后给你一个回答。但他有时候也会"一本正经地胡说八道"，因为他不是真的理解了，只是看多了类似的东西，觉得自己应该这么回答。

**要点：** AI的核心能力是"从大量数据中找到规律，然后根据规律做出预测或生成内容"。

### 机器学习

**比喻：** 就像教小朋友认猫。你不是给他一本规则手册说"四条腿、有尾巴、会喵喵叫的是猫"。你是给他看一万张猫的照片和一万张不是猫的照片，他自己就总结出来了。机器学习就是这个过程——让电脑自己从数据里学。

**要点：** 不需要手动写规则，给数据就行。

### 深度学习

**比喻：** 机器学习里面有个"尖子班"叫深度学习。普通的机器学习可能学会认猫和狗，深度学习能学会认上千种不同的动物，还能学会画画、写文章、跟你聊天。它用的"脑子"（神经网络）结构更复杂，层次更多，所以叫"深度"。

**要点：** 深度学习是机器学习的一种，特别擅长处理复杂的任务。

### 大语言模型（LLM）

**比喻：** 你用过手机上的输入法预测下一个词吧？你打"今天天气"，它会建议"很好"或"怎么样"。大语言模型就是这个功能的超级加强版——它不只是预测下一个词，它能预测接下来的一整段话、一篇文章、甚至一场对话。它读了互联网上几乎所有的文字，所以它能聊、能写、能分析。

**要点：** ChatGPT、Claude、文心一言，这些都是大语言模型。

### 生成式AI

**比喻：** 以前的AI就像一个选择题高手——给它选项，它能选出正确答案。现在的生成式AI更像一个创作高手——给它一个主题，它能写文章、画画、做表格、编代码，从无到有地"生成"新东西。

**要点：** 生成式AI能创造内容，不只是分析内容。

## 讲解节奏

1. 先从学员熟悉的东西引入："你用过手机上的语音助手吗？比如Siri？"
2. 引出AI概念，用比喻解释
3. 让学员知道：你现在正在跟一个AI聊天，这就是AI
4. 检查理解后进入下一个概念

## 常见误解纠正

- **"AI有意识/会思考"** → "AI其实不会'思考'，它更像是一个超级高级的文字接龙游戏。它根据看到的上下文，猜测最合适的下一个词是什么。"
- **"AI什么都知道"** → "AI知道很多，但也会犯错。有时候它会非常自信地说错误的信息，这叫'幻觉'。所以重要的信息你还是需要核实一下。"
- **"AI就是ChatGPT"** → "ChatGPT只是AI的一种应用。AI的范围大得多，比如你手机上的人脸识别、购物APP的推荐、导航路线规划，背后都是AI。"
